# node-content-sys(还未修改好20190429)
>node内容发布系统,界面布局统一使用响应式boostrap布局，ejs 数据渲染，后台(也是boostrap实现布局)实现node-content-sys的管理，数据库是使用mongodb实现存储。
## node-content-sys 小项目需求整理
1. 具备信息发布系统的特性
2. 具备后台管理的能力
3. 具备小型商品上架和销售的能力
4. 具备在线支付的能力
